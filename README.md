```md
# 🛍️ HAOLAT HUB – Elegant Hijabs & Accessories Website

A **visually stunning, fully responsive product showcase website** built with **HTML, CSS, and JavaScript**.  
Includes an **auto-scrolling product slider**, **light/dark mode**, and a **premium, luxury shopping experience** look & feel.

---

## ✨ Features

- 🎨 **Premium Design** – Gold-accented brand identity, clean typography, elegant layout
- 🌗 **Light & Dark Mode** – Smooth toggle with local storage persistence
- 🖼️ **Auto-Scrolling Product Slider** – Scrolls automatically every second, with arrows & dots for navigation
- 📱 **Responsive Layout** – Mobile-first design, works perfectly on all screen sizes
- 🖱️ **Hover Animations** – Soft zoom effects, shadows, smooth transitions
- 📖 **About Section** – Brand story & mission
- 📬 **Contact Section** – Simple contact form + WhatsApp, Instagram, and Facebook icons
- 🛒 **Product Grid & Modal** – Grid layout with clickable product cards that open detailed modal view
- 🎯 **Accessible & Keyboard-Friendly** – Supports arrow keys for slider navigation and escape key to close modals

---
```


---

## 🖥️ Customization

- **Brand Name & Logo:** Edit `.logo` text inside `index.html`
- **Colors:** Update `:root` CSS variables for `--deep-green`, `--gold`, etc.
- **Products:** Modify the `products` array in the `<script>` section (title, description, image)
- **Auto-Scroll Speed:** Change interval duration in `startAutoScroll()` (currently `1000ms`)
